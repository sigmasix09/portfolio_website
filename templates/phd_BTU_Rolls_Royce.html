Reseach Associate (PhD), BTU and Rolls Royce, Germany

(WIP) Topic: Virtual Engine Development Using Artificial Intelligence Methods

Sub-Topic: Data Compression and Reconstruction of High Fidelity CFD simulation using Neural Networks (NNs). The PhD is being performed at the Department of Numerical Mathematics and Scientific Computing in Brandenburg Techncial University (BTU), Cottbus, Germany. The industry partner is Rolls-Rolyce Deutschland (RRD), Dahlewitz office. The project is abbreviated as VTI-VI implying to the 6th phase of the project. The research project is funded by the state of Brandenburg, the Federal Government, and the EUropean Union (EU). It is a part of many large-scale projects and the overview can be found here VTI Project Overview.

The essence of the VTI-VI revolves around the efficient handling of immense high fidelity CFD simulation data using modern tools. The methodology development is based on the In-Situ In-Transit Hybrid Approach in ParaView from Kitware ( In Situ In Transit Hybrid Analysis using Catalyst-ADIOS2 and ParaView). The approach allows the CFD Simulation and Visulaization in parallel which are efficiently distributed on simulation node (CPU) and visulaization node (GPU), respectively. This approach promotes continuous Research & Development, scalabilty, high performance computing hardware (HPC) utilization, etc. The PhD work aims to integration modern tools like Machine learning (ML) and Neural Networks (NNs) to be for data compression, reconstruction, data storage, and etc. Therefore, developing a methodology that is uses modern reseach and tools, and can be integrated with the industrial infrastructure (RRD).
